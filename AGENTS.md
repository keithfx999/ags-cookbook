# AGENTS.md

This is a public AGS example repository.

Unless the task is explicitly about maintaining or contributing to the repository itself, treat the user as a consumer of AGS examples rather than a repository maintainer.

## For users of the examples

1. Read `README.md`
2. Read `examples/README.md`
3. Read the target example's `README.md`
4. Use the target example's local `Makefile` where available

## For contributors to the repository

If the task is explicitly about maintaining or contributing to the repository, read `CONTRIBUTING.md` first.
